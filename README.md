# 📊 Job Search Analytics

## Overview

Job Search Analytics is a data analysis project designed to organize, track, and analyze job opportunities from multiple job portals. The project helps job seekers monitor applications, identify hiring trends, and make data-driven decisions during the job search process.

This project demonstrates practical data analytics skills using Python, Excel, SQL, and data visualization techniques.

---

## Objectives

* Collect job listings from multiple platforms.
* Clean and preprocess job data.
* Analyze job trends and hiring patterns.
* Track applications efficiently.
* Visualize insights through charts and dashboards.

---

## Features

* Track job applications from multiple job portals
* Analyze required skills across job postings
* Compare opportunities by location and company
* Calculate candidate fit scores
* Monitor application status
* Generate summary reports and visualizations
